# Stable Diffusion Local Image Generation

This code uses the `stable_diffusion_cpp` built-in library for Python to run the Stable Diffusion image generation model locally.

## Getting Started

1. **Download the Model**
   - Recommended model: [Stable Diffusion v1.5](https://huggingface.co/second-state/stable-diffusion-v1-5-GGUF)

2. **Install Required Libraries**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Code**
   ```bash
   python3 main.py
   ```

## Model Parameters and Options

For details on model parameters and options, please refer to the [High-level API/Text to Image](https://github.com/william-murray1204/stable-diffusion-cpp-python) section in the `stable-diffusion-cpp-python` documentation.
